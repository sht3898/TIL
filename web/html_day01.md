## Dom Tree

태그 안에 태그가 있는 방식으로 구조화 되어 있는 것

```html
<body>
    <h1></h1>
    <ul>
        <li>HTML</li>
        <li>CSS</li>
    </ul>
</body>
```

* `<h1>`과 `<ul>`은 형제 관계
* `시맨틱 태그(Semantic Tag)`
  * `<div>` `</div>`
